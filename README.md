# Dance Academy Web Page

Welcome to the Dance Academy webpage project! This project is a website for a Dance Academy that showcases various dance styles, class schedules, instructors, and more. The website is built using HTML, CSS, JavaScript, Pug Template Engine, and Node.js to serve Static Files.

# Demo Video

## Features

- View information about different dance styles offered by the academy.
- Browse through the class schedules and Contact for dance classes.
- Contact the academy for inquiries and registration.

## Technologies Used

- HTML: Used for structuring the content of the webpage.
- CSS: Applied for styling and layout design.
- JavaScript: Implemented for interactivity and dynamic content.
- Pug: Used as a template engine to simplify HTML markup.
- Node.js: Used to serve the website and handle server-side tasks.
- Static Files: Various media files (images, videos) used in the website.

## Getting Started

1. Clone the repository:

   ```
   git clone https://github.com/AagamAJ/Dance-Academy-Web-Page-Using-HTML-CSS-JavaScript-NodeJS-PUG-as-Template-Engine-.git
   ```
2. Move to the Folder:

   ```
   cd projectDanceWebsite
   ```

3. Install dependencies:

   ```
   npm install
   ```

4. Start the server:

   ```
   node app.js
   ```

5. Open a web browser and navigate to `http://localhost` to access the Dance Academy webpage.

## Directory Structure

- `static/`: Contains static files such as images, videos, and CSS.
- `templates/`: Contains Pug templates for generating HTML pages.
- `app.js`: Main Node.js application file.
